# Advertisement List

Advertisement List is sample app, which fetch the Advertisement list form the server and show them as ListView.
Application uses the Architecture components,MVVM,Coroutines,Jetpack,Hilt etc. in Kotlin.

## Architecture
The app uses MVVM [Model-View-ViewModel] architecture to have a unidirectional flow of data, separation of concern, testability, and a lot more.
